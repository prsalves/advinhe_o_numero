# 🎯 Adivinhe o Número

Este é um projeto simples em Python onde o jogador precisa adivinhar um número gerado aleatoriamente pelo computador.

O programa escolhe um número dentro de um intervalo definido e o usuário tenta descobrir qual é, recebendo dicas se o número digitado é maior ou menor que o número correto.

## 🚀 Funcionalidades

* Geração de número aleatório
* Sistema de tentativas
* Dicas para ajudar o jogador (maior ou menor)
* Interação pelo terminal

## 🛠️ Tecnologias utilizadas

* Python

## 🎮 Como jogar

1. Execute o arquivo Python.
2. Digite um número quando o programa pedir.
3. O sistema informará se o número correto é maior ou menor.
4. Continue tentando até acertar o número.

## 📚 Objetivo do projeto

Este projeto foi desenvolvido para praticar conceitos básicos de programação em Python, como:

* Condicionais (`if`)
* Laços de repetição
* Entrada e saída de dados
* Uso de números aleatórios
